# 🚀 WSL2 Development Environment Dotfiles

My personal WSL2 development environment configuration files.

## What is Included

- zshrc - Zsh configuration with Oh My Zsh
- gitconfig - Git configuration
- tmux.conf - Tmux configuration
- pgpass - PostgreSQL passwords
- fzf.zsh - Fuzzy finder config
- start-services.sh - Start databases
- check-env.sh - Check environment

## Quick Start

Copy files to your home directory and reload shell.
